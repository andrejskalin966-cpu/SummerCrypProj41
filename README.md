main.cpp – точка входа чтение параметров, выбор и запуск алгоритма (надо определить список и формат параметров)
	
linearizing-set.cpp/.hpp - описание логики применения линеаризующих множеств

equation_system.cpp/.hpp - формат хранения MQS: отдельное описывание списка квадратичных уравнений, отдельное описание списка линейных уравнений; чтение MQS

linear_system.cpp/.hpp - методы решения линейных уравнений

quadratic_system.cpp/.hpp - методы решения квадратичных уравнений

search_tree.cpp/.hpp - определение логики решения MQS методом обхода дерева вывода (эвристика, backtracking (backjumping) и конфликтное ограничение + правила вывода)

encodings/*.mqs – папка с файлами кодировок MQS

*  необязательно

implication_graph.cpp/.hpp - для распространения литералов

test.cpp -  набор тестов для всех модулей

disjoint_set_union.cpp/.hpp - определение структуры, "система непересекающихся множеств" для множества литералов
